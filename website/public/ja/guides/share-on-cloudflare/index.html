自分の Cloudflare で、無料の共有を

Cmd+Shift+L で文書がウェブページとして公開される — KaTeX、Mermaid、ダークモード、モバイル対応。仕掛けはこうだ。公開先は我々の Cloudflare アカウントではなく、あなたのアカウント。個人のワークロードなら無料枠で楽に収まる。

なぜセルフホストの共有か

あなたがこれまでクリックしたすべての「共有」ボタンは、文書を他人のサーバーに、他人の規約の下、他人の寿命でアップロードしてきた。note.md の共有プラグインは、小さな Worker をあなたの Cloudflare アカウントにデプロイする。あなたのリンク、あなたのデータ、あなたのキルスイッチ。無料枠（10万リクエスト/日）は、人間が文書を共有する量をはるかに超えている。

10分でデプロイ

cd worker
pnpm install
wrangler login
wrangler kv:namespace create SHARES     # copy the id into wrangler.toml
openssl rand -hex 32 | wrangler secret put SHARE_API_KEY
wrangler deploy                          # prints your Worker URL

Worker の URL と API キーを note.md → Preferences → Share に貼り、再起動、完了。詳細はリポジトリの worker/README.md にある。

手に入るもの

  • キー一発：Cmd+Shift+L で現在のファイルを公開。URL はクリップボードに入る。再共有でその場で更新、共有解除で 410 を返す。
  • 忠実なレンダリング：KaTeX の数式、Mermaid 図は SVG に、シンタックスハイライト、prefers-color-scheme によるライト／ダーク、モバイル最適化。
  • 画像も込み：画像の多い文書は、自動的に Cloudflare R2（これも無料枠）にあふれ出す。
  • エージェント対応：Worker は MCP エンドポイントを公開しているので、エージェントが代理で公開できる — notemd -s draft.md はどんなスクリプトからでも動く。

FAQ

いくらかかるのか？

個人利用ならゼロだ。Cloudflare の無料枠には1日10万 Worker リクエストと 10GB の R2 ストレージが含まれる — 人間が文書を共有する量の、数桁上だ。

共有したページを取り下げられるか？

できる、即座に。File → Unshare（または notemd share --unshare）でリンクは失効し、訪問者には 410 が返る。あなたの Worker なのだから、丸ごと消してしまってもいい。
